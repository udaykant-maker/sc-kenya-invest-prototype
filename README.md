# SC Kenya Invest — Mobile Prototype

Interactive mobile prototype for Standard Chartered Kenya's equities trading feature, built as a WebView for SC Mobile (netbanking + mobile banking).

## Live Demo
[View on GitHub Pages →](https://udaykant-maker.github.io/sc-kenya-invest-prototype/)

## What's included
- **Home screen** — Portfolio value card, quick actions, NSE/US market toggle
- **Stock Detail** — Live-style price chart (1D/1W/1M/3M/1Y), key stats, company info
- **Order Ticket** — Buy/Sell, Market/Limit/Stop orders, CASA account debit, brokerage + stamp duty calc
- **Confirm Sheet** — Bottom sheet confirmation → success state
- **Portfolio** — Holdings with avg price and P&L
- **Discover** — Search, Top Movers, NSE + US stock list

## Brand
Follows Standard Chartered brand guidelines:
- SC Teal (#00848F) as primary colour
- SC Green (#00843D) for positive/CTA
- SC Cobalt (#1B3A6B) for portfolio hero
- Source Sans 3 typography
- SC three-bar flag mark logo

## Backend
Order execution powered by **Vault22** · Supports NSE Kenya + US (NYSE/NASDAQ) equities.
